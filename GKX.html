Go-Karting Xtreme

A Roblox roleplay Go-Karting game created by Humza. This game was created and updated by Humza between March 2017 to September 2019

About Go-Karting Xtreme

Go-Karting Xtreme is a roleplay Go-Karting game that is ran on the Roblox game engine. It is a game that aims to provide a realistic experience of a real life Go-Karting track, including having a reception, a breifing room and a track area. Players can apply for different roles, ranging from track staff to racing host.

The game was founded and owned by Humza between March 2017 to September 2019. The game was then sold for £100 to MarketManager1.

The Issue

After playing a few Go-Karting roleplay games on Roblox myself during late 2016, I noticed that most of the games had very similar issues. These issues include:

• Bad customer service
• Unclear racing session times
• Unclear track rules
• Constant need to develop new tracks

The Solution / Key Features

During Early 2017, I decided that I would attempt to solve these issues. I did this by looking at all the main issues and trying to figure out how to find a solution by looking at how a real life Go-Kart track runs. These are the solutions I found:

• Create a Reception area where players can ask questions
• Create a in-game screen system to clearly show racing session timings
• Create a briefing room to explain track rules
• Make the track using a modular system, allowing for new tracks to be developed quickly, while also making the track automatically change after each session

Other key features of the game includes:

• A very functional control room to control pit gates and track light system
• Automatic breifing room and track entrance/exit door system
• A Multi story track
• Player staff role displayed above players head
• A mobile friendly start menu

About Rosebridge

Rosebridge is a city roleplay game, where you can work at a variety of different jobs, own a house and drive a car. The game is created using the "Roblox Studio" game engine. Some of the places you can work at include fast-food restaurants such as fried chicken shop and a pizza place. Alternatively, players can purchase a "Gamepass" through the Roblox currency "Robux" to get access to additional access to special roles and features. These game passes are:

• Premium house
• Premium cars pack
• Police role
• Fireman role
• In-car radio
• Car colour customization

The game was created in a team called "Dynamic Games", where I was the main scripter for the game during Late 2017.

My contribution

My contribution to the game was to create some of the essential scripts required for the main features of the game before it was set to release at around December 2017. 

The first main feature that I worked on was the start menu. The features for the menu includes a loading screen, a fade in and fade out animation of the "Dynamic Games" logo and the main menu selection. The selections included are "start", "choose team" and "frequently asked question" buttons. When any of the buttons are selected, an animation on the user interface is used to allow a smooth transition to the selected option.

The second feature that I worked on was the vehicle selection menu. This included creating a user interface to show the player what cars they can spawn into the game, while also allowing players who purchased some of the game passes to spawn in special vehicles. This can include high end, police and fireman vehicles.

The third feature that I worked on was the house ownership feature. This included allowing the player to click a button to own the house, but to also make sure they can only own one house in the game. Other features of house ownership also include ensuring that the player can open their house main door and all garage doors.

In addition to this, other small features that I worked on for the game includes:

• The vehicle colour customisation system
• The "GIve Food" system for fast food restaurants
• Helping to bug fix the fire system

Artificial Intelligence

A Genetic Algorithm that I created using C#. This was part of my Artificial Intelligence module during the first semester of second year University (Late 2019).

About Genetic Algorithm

The task set was to create a genetic algorithm that would aim to improve a population of weights, each containing 60 individual weights. That means that a population of 100 members would have 100 X 60 Weights. The skeleton code is given to us also has a GetResults function, which accepts an array of 60 individual weights. This function returns a number showing if the weight is good or not, ranging from 0 (bad) to 1 (good). The algorithm will then go through the following steps to improve itself:

• Selection: This is where a few sets of weights are randomly selected. For example, if the selection size is set to 4, it will select a random 4 X 60 weights from the population of 100 X 60 weights. After this, it will then look through the 4 X 60 weights and run a GetResults function on all 4 sets of weights. Selection will then get the best 2 sets of weights and store them in a list called parents.

• Crossover: Crossover is when you combine the first 30 weights parent 1 and the last 30 weights of parent 2. Alternatively, the program can also select the fittest weight instead of combining both parents, depending on the crossover rate. This becomes the child.

• Mutation: Mutation is when you go through the child individual weights, and depending on the mutation rate, the program will replace the weight with a new weight.

• Generations: The mutated child is now stored in a new population. The selection, crossover and mutation are then repeated until the new population size is the same as the initial population size. The initial population is then replaced with the new population, and the new population list is removed. This is one generation complete, and the entire process is repeated to complete another generation. After each generation, the fittest set of weights in the algorithm should increase.

The Report

After coding the Genetic Algorithm, I was also tasked with creating an experiment to see what happens when I change some of the genetic algorithm settings. This can include changing the population size, selection size, mutation rate, crossover rate or generations. In my report, I decided to test the mutation rate. After running the experiment several times, I found that the mutation rate of 0.025 resulted in a better set of weights when compared to the mutation rate of 0.05.
